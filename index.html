Explora las Tecnologías Web

Aprende los fundamentos de HTML, CSS, JavaScript y Python con ejemplos claros y prácticos.

Comenzar

HTML (HyperText Markup Language)

HTML estructura el contenido de una página web: define títulos, párrafos, imágenes, enlaces y más.

Principales etiquetas:
  • <h1> — Título principal.
  • <p> — Párrafo.
  • <a> — Enlace a otra página.
  • <img> — Imagen en la página.
Ejemplo:
<h1>Hola Mundo</h1>
<p>Bienvenido a mi primera página web</p>
<a href="https://www.example.com">Visitar enlace</a>

CSS (Cascading Style Sheets)

CSS define el estilo visual de los elementos HTML: colores, tamaños, márgenes y tipografía.

Principales propiedades:
  • color — Cambia el color del texto.
  • background-color — Define el color de fondo.
  • font-size — Ajusta el tamaño del texto.
  • margin / padding — Controla los espacios.
Ejemplo:
p {
  color: blue;
  background-color: lightgray;
  font-size: 18px;
  padding: 10px;
}

JavaScript

JavaScript da vida a las páginas web, permitiendo responder a acciones del usuario y modificar el contenido dinámicamente.

Principales funciones y estructuras:
  • document.getElementById() — Busca un elemento HTML.
  • addEventListener() — Escucha eventos.
  • if / else — Condicionales.
  • function() — Define una función.
Ejemplo:
<button onclick="saludar()">Haz clic</button>
<script>
function saludar() {
  alert("¡Hola desde JavaScript!");
}
</script>

Python

Python es un lenguaje versátil, usado en desarrollo web, análisis de datos, inteligencia artificial y más.

Principales estructuras:
  • print() — Muestra texto en pantalla.
  • for / while — Bucles.
  • def — Define una función.
  • if / elif / else — Condicionales.
Ejemplo:
def saludar(nombre):
    print(f"Hola {nombre}, bienvenido a Python!")

saludar("Marcelo")
